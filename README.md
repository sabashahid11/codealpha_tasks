# CodeAlpha Tasks - Python Projects

This repository contains 3 Python projects created for CodeAlpha internship tasks.

## Projects

### 1. Stock Portfolio Tracker (stock_tracker.py)
A program to track stock investments and calculate total portfolio value.

**Features:**
- Track multiple stocks with quantities
- Calculate total investment
- Save portfolio to text file
- Menu-driven interface

### 2. Rule-Based Chatbot (chatbot.py)
A simple interactive chatbot with predefined responses.

**Features:**
- Responds to greetings and basic questions
- Random response selection
- Simple conversation flow
- Exit command

### 3. Hangman Game (hangman.py)
A graphical word guessing game built with Tkinter GUI.

**Features:**
- GUI interface with Tkinter
- Random word selection from a word list
- Hint system to reveal letters
- Visual feedback for correct/incorrect guesses
- Win/loss detection with congratulatory popup
- Play again functionality
- Colorful and user-friendly interface
